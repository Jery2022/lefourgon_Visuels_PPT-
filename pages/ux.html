Méthodologie UX — Mobile First absolu

La démarche UX de Le Fourgon suit un processus en 4 étapes séquentielles : exploration utilisateur → zonings → wireframes → prototypes interactifs. Chaque étape est d'abord réalisée sur mobile (375px), puis déclinée en tablette (768px) et desktop (1280px+).

1
Exploration utilisateur
Analyse des 3 personas (Sophie, Clara, Thomas). Cartographie des parcours BtoC et BtoB. Identification des points de friction sur le site actuel.
Sem. 3 · Figma + UserTesting
2
Zonings
Découpage en blocs fonctionnels. Hiérarchie visuelle. Attribution des rôles par zone. Mobile 375px en premier, puis tablette et desktop.
Sem. 3 · Figma low-fi
3
Wireframes
Maquettes basse fidélité avec contenu réel. Structure de navigation, composants UI et flux de données. Annotés pour le handoff dev.
Sem. 4 · Figma
4
Prototypes interactifs
Navigation cliquable entre écrans. Tests des 3 parcours clés : achat BtoC, devis BtoB, dashboard impact. Tests avec utilisateurs réels.
Sem. 5–6 · Figma + Maze
Principes UX fondateurs
Mobile First absolu
375px en premier. Aucune maquette desktop sans mobile validé au préalable.
Double parcours étanche
Bifurcation BtoC/BtoB dès la homepage. Tunnels distincts sans mélange.
Impact visible above-the-fold
Compteur de bouteilles évitées visible sans scroll sur mobile.
Friction minimale
Tunnel BtoC en 3 étapes max. Devis BtoB en moins de 5 minutes.
WCAG 2.1 AA obligatoire
Contraste 4,5:1 min. Zones tactiles 44px. Labels hors placeholder.
Test sur iPhone SE
Validation systématique sur le plus petit format cible (375px).

Adaptation responsive — Mobile → Tablette → Desktop

Chaque composant est conçu en 375px et adapté progressivement. Les règles d'adaptation sont définies pour chaque breakpoint via les Design Tokens Figma.

Breakpoints et adaptations
Mobile — 375px
1 colonne · Header vert plein · Bottom nav · Scroll vertical · CTAs pleine largeur · Ratio images 4:5
Tablette — 768px
2–3 colonnes · Header blanc + nav partielle · CTAs côte à côte · Sidebar optionnelle · Grille flexible
Desktop — 1280px+
4 colonnes · Header blanc nav complète · Layout 60/40 · Sidebar filtres · Portail B2B complet
